# Nature Style — 语音克隆页面设计规范

## 色彩系统

| Token | 值 | 用途 |
|-------|-----|------|
| 主色 | #2E7D32 (Forest Green) | 按钮、链接、进度条 |
| 主色浅 | #E8F5E9 | 卡片背景、hover |
| 背景 | #FAFAFA | 页面底色 |
| 文字主 | #1B1B1B | 正文 |
| 文字辅 | #666666 | 说明文字、placeholder |
| 边框 | #E0E0E0 | 输入框边框 |
| 错误 | #D32F2F | 校验失败 |
| 成功 | #388E3C | 克隆完成 |

## 排版

- 字体: system-ui, -apple-system, sans-serif
- 标题: 24px/600, 行高 1.3
- 正文: 16px/400, 行高 1.6
- 代码: 14px "JetBrains Mono", monospace

## 间距

- 页面最大宽度: 720px, 水平居中
- 区块间距: 32px
- 表单元素间距: 16px
- 卡片内边距: 24px

## 组件风格

- 输入框: 圆角 8px, 边框 1px solid #E0E0E0, focus 时边框变为 #2E7D32
- 按钮: 圆角 8px, 高度 44px, 主按钮填充 #2E7D32 白色文字
- 卡片: 白色背景, 圆角 12px, 阴影 0 2px 8px rgba(0,0,0,0.06)
- 进度条: 高度 6px, 圆角 3px, 填充色 #2E7D32

## 整体基调

自然、干净、专业。不要花哨动画。以内容为中心，让用户专注于声音克隆任务。
